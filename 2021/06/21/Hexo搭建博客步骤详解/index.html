五、常用命令汇总

命令 用途
node -v 查看 node 版本
npm -v 查看 npm 版本
cnpm -v 查看 cnpm 版本
hexo -v 查看 hexo 版本
mkdir “” 创建目录
cd “” 切换路径
hexo c 清理博客
hexo g 生成
hexo s 启动博客
hexo d 部署到远端（如 GetHub）
